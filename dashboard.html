How SmartExam works

Add student roll numbers → Allocate seats using randomisation (Fisher-Yates shuffle) → Generate a sorted collection sheet for invigilators. Seating is randomised to prevent malpractice; the collection sheet is sorted by roll number for easy administration.

Reset Data

🗑️ Reset Student List

This will remove all student records and clear seat allocations. Use this to start a new exam session with a fresh student list.

🔴 Reset All Data

This will clear everything including student records, seat allocations, and all saved state. This action cannot be undone.

ℹ️
All data is stored in your browser's localStorage. Data persists between sessions until you explicitly reset it. Clearing browser data will also remove saved records.
